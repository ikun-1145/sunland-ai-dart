# Sunland AI App

🚀 基于 Flutter 开发的 AI 助手应用

## ✨ 项目简介
Sunland AI 是一个轻量、现代化的 AI 应用，支持：
- 🤖 AI 对话
- ✍️ 文本优化 / 润色
- 📊 内容分析
- 🔄 多场景智能交互

本项目使用 **Flutter + Dart** 开发，目标是打造一个跨平台的 AI 客户端。

---

## 🛠️ 技术栈
- Flutter
- Dart
- （后续可接入）Supabase / OpenAI API

---

## 📱 运行方式

确保你已经安装：
- Flutter SDK
- Dart 环境

运行项目：

```
flutter pub get
flutter run
```

---

## 📂 项目结构（简要）

```
lib/        # 主代码
assets/     # 资源文件
android/    # Android 平台
ios/        # iOS 平台
```
